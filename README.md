College Student Management Database (SQL Practice)

This project contains SQL practice tasks for creating and managing a student database.

Topics covered:
• Creating a database and tables
• Altering table structure
• Inserting records
• Updating and deleting data
• Writing SELECT queries with conditions
• Sorting and filtering data

SQL concepts practiced:
DDL – CREATE, ALTER, DROP  
DML – INSERT, UPDATE, DELETE  
DQL – SELECT, WHERE, LIKE, BETWEEN, ORDER BY  

Total tasks completed: 30

This practice helped me strengthen my SQL skills and made me more confident in writing SQL queries.
